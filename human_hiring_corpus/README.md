# Human hiring corpus — scripts & prompts

The exact prompts participants recorded for the human corpus
([`multispeak/hiring-accent-speech-human-voices`](https://huggingface.co/datasets/multispeak/hiring-accent-speech-human-voices)).

| File | Contents |
|---|---|
| `scripted_scripts.csv` | The 4 scripted scenarios (`category, script`) — every speaker reads the same text, so score differences isolate delivery. |
| `unscripted_prompts.csv` | The 2 open-ended prompt sets (`category, question_number, question`) used to elicit natural, unscripted speech. |

## Recording instructions (given to participants)

- Record in a **quiet space with minimal echo**; on a phone, hold it **6–8 inches away** to avoid popping.
- Read the scripted scenarios **aloud and naturally**, as if speaking in a real job interview or with a colleague —
  fewer filler words help comparability, but recordings need not be perfect.
- Save each file by the initials of the script name (e.g. `personal-introduction` → `pi.mp3`).

The four scripted scenarios are **personal introduction, personal commitment, financial product, and client
disagreement**; the two unscripted sets are a **professional introduction** and a **recent disagreement**.
